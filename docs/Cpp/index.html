C++ 笔记

C++ 是一门高效、强大、贴近底层的编译型高级语言，能够承担包括网络、系统或硬件开发、科学计算等多种任务。本文是我在学习现代 C++ 时的笔记，旨在让自己能够回顾学习过的内容——毕竟 C++ 的内容比较繁多。
