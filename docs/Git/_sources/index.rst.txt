Git 使用笔记
==============

本文的绝大部分内容来自 Git 的官方文档。官方提供了中文译本 `Git 文档简体中文版 v2 <https://git-scm.com/book/zh/v2>`_ ，可以在线阅读或者下载 pdf 进行浏览。

通过版本控制系统（Version Control System, VCS）对文件的每个历史版本进行管理，是工程开发中不可缺少的模式。VCS 允许管理者方便地回退版本、多人协作，借此达成对项目的开发、维护与管理。

Git 可能是当下最流行的 VCS。如果你是一个编程人员或者 Github 用户，你必须会使用 Git。
